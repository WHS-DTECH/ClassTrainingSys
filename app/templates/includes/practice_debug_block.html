Practice: Where Would You Debug?

  1. Example 1: 
    def greet(name):
    print("Hello" + name)

greet()
    Solution: The function greet() is called without an argument, but it requires one. This will cause a TypeError.
  2. Example 2: 
    numbers = [1, 2, 3, 4, 5]
for i in range(6):
    print(numbers[i])
    Solution: The loop goes from 0 to 5, but numbers only has 5 elements (indices 0-4). numbers[5] will cause an IndexError.
  3. Example 3: 
    def divide(a, b):
    return a / b

print(divide(10, 0))
    Solution: divide(10, 0) will cause a ZeroDivisionError because division by zero is not allowed.
  4. Example 4: 
    total = 0
for i in range(5):
    total = total + i
print(totals)
    Solution: The variable totals is not defined; it should be total.
